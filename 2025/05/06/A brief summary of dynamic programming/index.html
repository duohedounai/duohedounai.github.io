A Brief Summary of Dynamic Programming

Posted on 2025-05-06 Edited on 2025-06-02 Views: Word count in article: 686 Reading time ≈ 2 mins.

Algorithm Description

Dynamic programming is mainly used to solve multi-stage decision-making problems. Solving a problem using dynamic programming usually involves the following steps:

  1. Divide the original problem into several stages, each stage corresponds to several sub-problems, and extract the features (states) of these sub-problems.
  2. Find the possible decisions for each state, or the transition between states (state transition equation).
  3. Solve the problems in each stage in sequence.

Problems that can be solved using dynamic programming must meet the following characteristics:

  • 最优化原理（最优子结构性质）：原问题的最优解包含了子问题的最优解。
  • 无后效性
  • 子问题的重叠性：这个不是必要条件，但可以体现动态规划的优势。
