Reading Notes-Programming Pearls

Posted on 2025-03-30 Views: Word count in article: 224 Reading time ≈ 1 mins.

Dynamic Window Approach (DWA) is a commonly used local path planning method.

Common global path planning algorithms such as A* and Dijkstra are state sampling algorithms, while the DWA algorithm is an action sampling algorithm.
